Insert


Insert clause is a clause in SQL used to insert record into a database.
The standard format of insert clause is the following:
INSERT INTO table name (column names) VALUES (values);

An example of a INSERT clause:

INSERT INTO Workers (firstname, lastname, age) VALUES ('Bob', 'Michaelson', 25);
With this query, we insert new record into Workers table, which firstname will be Bob, lastname will be Michaelson and age will be 25.
