Select


Select clause is a clause in SQL used to retrieve data from database.
The standard format of select clause is the following:
SELECT column names FROM table name;

Two examples of a SELECT clause:

SELECT * FROM Workers;
SELECT firstname, lastname, age from Workers;
In the first query result, we want to see ALL columns' values of Workers table, * mark stands for it.
In the second query result, we want to see only firstnames, lastnames and ages column values.
