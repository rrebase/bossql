Introduction


SQL is a standard language for accessing and manipulating data held in relational database management system (RDBMS).
SQL stands for structured query language.
It has several features such as:
  • execute queries against database.
  • retrieve data from a databases.
  • insert a record into a database.
  • update records in a database.
  • delete records from a database.
  • create new database.
  • create new tables in a database.
  • set permissions on tables, procedures and views.
