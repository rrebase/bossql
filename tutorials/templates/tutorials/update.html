Update


Update clause is a clause in SQL used to change records data in a database.
The standard format of insert clause is the following:
UPDATE table name SET column name = value;

An example of a UPDATE clause:

UPDATE Workers SET age = 27;
With this query, we change ages of all the records in Workers table to 27.
In order to change only Bob Michaelson's age, we have to add WHERE, as following:
UPDATE Workers SET age = 27 WHERE firstname = 'Bob' AND lastname = 'Michaelson';
It selects only the record, where worker's firstname is Bob and lastname is Michaelson.
