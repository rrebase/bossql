Create


Create clause is a clause in SQL used to create a new database or table.
The standard format of create clauses are the following:
CREATE DATABASE database name;
CREATE TABLE table name(columns and their descriptions); Examples of a CREATE clause:

CREATE DATABASE newDB;
CREATE TABLE Workers (

    workerId INT AUTO_INCREMENT PRIMARY KEY,
    firstname VARCHAR(80),
    lastname VARCHAR(140),
    age SMALLINT(3)
);
In the first query, we create new database named newDB.
In the second query, we create new table named Workers that has the following columns.
