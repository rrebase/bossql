About

bossql is a place to learn SQL through a wide range of different challenges. From simple select all columns to complex queries with many components. It was made as a software engineering project for a course at TTU.

FAQ

Frequently Asked Questions
What is SQL?

SQL (Structured Query Language) is a language used for managing data held in a relational database. This includes things like selecting or updating records that match given criteria. SQL databases are the backbone of many applications which need to handle large data sets.

Why is it called bossql?

You have to be the boss of SQL to beat all the challenges in every topic.

Which SQL engine is used in challenges?

PostgreSQL 9.5

Why do I have to log in?

To save and track your progress we need to have a way to identify you. User data is not shared with anyone.
